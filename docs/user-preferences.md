## Key Discoveries

### 1. Saxophone as Texture (Not Melody)

**Discovery**: Prompt 41's saxophone works because it's **atmospheric background**, not a melodic lead.

**Why This Matters**:
- Melodic leads demand conscious attention
- Atmospheric textures provide timbral variety without distraction
- Prevents habituation while maintaining focus-friendly character

**Hypothesis for Future Testing**:
- Try other woodwinds as texture: clarinet, flute, oboe
- Try brass as texture: muted trumpet, French horn
- Keep instruments in reverb-soaked background, never foreground

---

### 2. Swing Quantization Works

**Discovery**: Prompt 40's swing quantization adds human groove without overwhelming.

**Why This Matters**:
- Perfect grid quantization can feel mechanical
- Swing adds Brazilian/jazz feel without complexity
- Works especially well for midday refresh (novelty through groove variation)

**Hypothesis for Future Testing**:
- Apply swing to more house/techno grooves
- Test different swing amounts (subtle vs. pronounced)
- Combine with other Brazilian elements (samba, bossa nova)

---

### 3. Quality Variance is Normal

**Observation**: Suno generates 2 versions per prompt. Quality varies significantly between them.

**Pattern**:
- Typical: 1 excellent + 1 mediocre
- Sometimes: 2 excellent (lucky)
- Rarely: 2 mediocre (re-generate)

**User Strategy**:
- Always review both versions
- Keep only the excellent one
- Don't re-generate unless both are poor
- This is expected behavior, not a bug

## Rating Guidelines

### When to Rate ⭐

Rate a prompt excellent if it:
1. Effectively induces intended brain state
2. Maintains focus without demanding attention
3. Doesn't cause habituation/boredom
4. Has timbral/textural interest
5. Feels appropriate for its time block

### Rating Format

Add to CSV "Rating" column:
- Short: `"Excellent ⭐"`
- With context: `"Excellent for midday refresh ⭐"`
- With discovery: `"Excellent - sax texture perfect for midday refresh! ⭐"`

### What NOT to Rate

Don't rate based on:
- Personal music taste preferences
- Genre preferences (e.g., "I don't like jazz")
- Comparison to non-focus music
- Whether you'd listen to it recreationally

Focus music is **utilitarian** - rate based on effectiveness for programming work, not entertainment value.
