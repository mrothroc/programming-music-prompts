## Instrument Palette

**Core instruments** (proven parent DNA):
- Analog synthesizers (Juno-60, Prophet-5, OB-8, MS-20)
- Electric piano (Rhodes, Wurlitzer)
- Sub bass (sine/triangle waves)
- Minimal percussion (four-on-floor, boom bap, brushes)

**Cultural instruments** (textural additions):
- African: balafon, kora, thumb piano, talking drums, kalimba
- Brazilian: nylon guitar, pandeiro, tamborim, surdo, cuica
- Appalachian: fingerpicked guitar, banjo

**Mutation instruments** (new additions):
- Mellotron (orchestral tape textures)
- EBow guitar (sustained electric tones)
- Lap steel guitar (sliding sustained tones)
- Marimba (wooden melodic percussion)
- Saxophone (atmospheric texture, not melodic)
